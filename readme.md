# 📈 Thông Tin Thị Trường - Market Insights Dashboard

Dự án trang web tổng hợp dữ liệu thị trường, tin tức và thời tiết. Hệ thống được thiết kế tập trung vào trải nghiệm người dùng trên máy tính cá nhân.

> ⚠️ **LƯU Ý:** Trang web hiện tại được tối ưu hóa tốt nhất để hiển thị trên **Laptop/Desktop** và chạy trong môi trường **Local (Máy cục bộ)**.

## ✨ Tính năng chính
- **Dashboard YouTube:** Cập nhật số liệu (Subscribers, Views) từ các kênh lớn qua API.
- **Tin tức 24/7:** Hiển thị tin tức mới nhất từ các nguồn uy tín.
- **Dự báo thời tiết:** Theo dõi nhiệt độ tại các thành phố lớn (Hà Nội, Tokyo, Seoul...).
- **Hệ thống Auth:** Giao diện Đăng nhập/Đăng ký hiện đại.

## 🛠 Công nghệ sử dụng
- **Frontend:** HTML5, Tailwind CSS, Lucide Icons.
- **Data:** Kết nối qua Fetch API (api2.js).
- **Trạng thái:** Phiên bản thử nghiệm (Local Development).

## 🚀 Hướng dẫn mở trang web (Dành cho Developer)

Để trang web hoạt động đầy đủ tính năng (đặc biệt là các script JavaScript), bạn **không nên** mở trực tiếp file HTML bằng trình duyệt. Hãy sử dụng **Live Server** theo hướng dẫn sau:

### 1. Cài đặt môi trường
1. Mở thư mục dự án bằng **Visual Studio Code (VS Code)**.
2. Vào mục **Extensions** (phím tắt `Ctrl + Shift + X`).
3. Tìm kiếm và cài đặt Extension tên là **"Live Server"** của tác giả *Ritwick Dey*.

### 2. Cách khởi chạy
* **Cách 1:** Chuột phải vào file `index.html` chọn **Open with Live Server**.
* **Cách 3:** Nhấn vào nút **Go Live** ở thanh trạng thái (góc dưới cùng bên phải màn hình VS Code).

### 3. Truy cập
* Trình duyệt sẽ tự động mở địa chỉ: `http://127.0.0.1:5500/index.html`
* **Lưu ý:** Giao diện được thiết kế chuẩn cho màn hình Laptop. Nếu xem trên điện thoại, một số thành phần có thể chưa được tối ưu hoàn toàn.
